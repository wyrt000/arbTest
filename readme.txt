          arbTest 基金套利监控看板与流水线系统 - 快速使用指南 (V8.5)
================================================================================

一、 系统简介
本系统是一个工业级的跨市场基金折溢价套利监控系统。
项目采用“大一统底层基座 (arbcore) + SQLite WAL并发数据库 + ArbDashboard 核心看板”架构，
能够实时监控黄金原油、QDII 欧美、QDII 亚洲、国内 LOF、白银等各类上市基金的实时溢价与估值。

--------------------------------------------------------------------------------
二、 极速启动
在主目录下，双击运行以下批处理文件即可一键启动：
  ▶ start_dashboard.bat  (一键启动后端 8000 端口服务 + 前端 5173 页面监控服务)

* 注：浏览器会自动打开网页看板 (http://localhost:5173/)。

--------------------------------------------------------------------------------
三、 各数据源关系与功能说明 (四级瀑布流设计)

程序在获取行情时会自动采用降级和备选机制，不需要同时开启所有交易软件：

1. 国内 A 股及期货行情 (四级瀑布流)：
   * 银河 QMT (最推荐，Socket 直连，速度最快)
   * 通达信 TQ 接口 (内存直连，需要启动通达信客户端并配置 TQ 插件)
   * 国金 QMT (xtquant 极速策略库)
   * 新浪 API (低频轮询，无需运行任何客户端，自动作为最终兜底)

2. 美股/港股标的实时现价 (双通道)：
   * 盈透证券 (IB Gateway / TWS)：美股 ETF 夜盘/实时主通道。
   * 富途 OpenD (FUTU)：备用通道，若无 IB 账户，开启富途牛牛 API 后自动接管行情。

3. 历史净值与估值数据 (云端备份 + 本地爬虫)：
   * Woody API：估值计算的核心锚点与历史定价基准。
   * 腾讯/新浪接口：国内 LOF 与美股历史 K 线/成交量的自动化获取。

--------------------------------------------------------------------------------
四、 隐私安全防护说明
本系统已严格配置 .gitignore 规则，以下涉及个人账号隐私的文件已被绝对屏蔽，绝不会上传至 GitHub：
1. 个人账户密码文件：`arbcore/config/account_private.py` (存储您的 Woody、银河、国金、VPS 等账号秘钥)
2. 数据库敏感流水表：`database/arb_master.db` (只保留结构模板，排除交易流水)
3. 临时和本地缓存目录：`logs/`, `data/`, `scratch/` 

在 push 提交代码时，请确保本地修改没有强行将上述敏感文件加入追踪队列。
